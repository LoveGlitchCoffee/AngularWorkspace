BlockAge

BlockAge, algorithm visualisation but its a game

This game was the product of my second year at university team project module. The game visualises 3 search algorithms (Depth First, Breadth First and A*) and 3 sort algorithms, (Bubble, Selection and Insertion). The game features a tower defense / reversed time attack gameplay, lightweight and intuitive animations for algorithm visualisation and map editor.


My Story

The tasks given to us in the team project was to create an algorithm visualisation system. I was dissapointed as my expectations was a multiplayer game. Luckily my team agreed that we could do something fun, but still follow the specifications, and BlockAge was the result of that (no multiplayer though). I did get to pick up some pixel art experience along the way.
